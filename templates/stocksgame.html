I made a stock market game using historical data because one thing I like to daydream about is going back in time and getting so rich that I don't need a job. The stack is basically the same as BasketballGM which is typescript and IndexedDB

It has all the data for SP500 stocks at open and close, along with BTC and ETH, along with some international ones such as Rheinmetall and some ETFs as well. Let me know if I should add anything else.

You can play it at https://mitchypi.github.io/newgame/
